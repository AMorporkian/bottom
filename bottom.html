bottom.event module¶

class bottom.event.EventsMixin(getparams: typing.Callable, *, loop: asyncio.base_events.BaseEventLoop) → None[source]¶

Bases: object

on(event: str) → typing.Callable[source]¶

Decorate a function to be invoked when the given :param:`event` occurs. The function may be a coroutine.

A decorator that takes a function and registers it with the event.

Example:

import asyncio

events = EventsMixin({'test': ['arg', 'one', 'two']})

@events.on('test')
def func(one, arg):
    print(arg, one)

event = 'test'
kwargs = {'one': 1, 'two': 2, 'arg': 'arg'}

events.trigger(event, **kwargs)
loop = asyncio.get_event_loop()
# Run all queued events
loop.stop()
loop.run_forever()
Parameters:event – The name of the event to wait for.
Returns:The wrapped function.
